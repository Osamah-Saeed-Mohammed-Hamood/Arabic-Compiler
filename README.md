# Arabic-Compiler

## نظرة عامة
المترجم العربي هو مشروع يهدف إلى تطوير لغة برمجة جديدة تعتمد على المفردات العربية، مما يتيح للمبرمجين كتابة الأكواد بلغتهم الأم. تم بناء المترجم من الصفر، بدءًا من التحليل اللفظي وصولاً إلى توليد الكود وتنفيذه على آلة افتراضية خاصة.

## المميزات
- كتابة الكود بالعربية: استخدام كلمات مفتاحية عربية مثل برنامج، اذا، كرر، و طالما.
- دعم لأنواع البيانات الأساسية: صحيح، حقيقي، منطقي، حرف، و سلسلة.
- هياكل تحكم متكاملة: جمل شرطية (اذا ... والا) وحلقات تكرار (كرر، طالما).
- دعم للدوال والإجراءات: إمكانية تعريف واستدعاء الدوال مع تمرير المعاملات.
- آلة افتراضية (VM): بيئة تشغيل مخصصة لتنفيذ الكود المترجم بكفاءة.

  ## كيفية الاستخدام
  1- كتابة الكود: قم بإنشاء ملف نصي بامتداد .ar واكتب برنامجك باستخدام قواعد اللغة العربية.
  2- الترجمة: استخدم المترجم لتحويل ملف .ar إلى كود وسيط.
  3- التنفيذ: قم بتشغيل الكود الوسيط باستخدام الآلة الافتراضية المرفقة.

  ## هيكلية المشروع
  - ArabicCompiler.Core: يحتوي على المنطق الأساسي للمترجم (Lexer, Parser, Semantic Analyzer, Code Generator).
  - ArabicCompiler.Runtime: يحتوي على الآلة الافتراضية (VM) لتنفيذ الكود.
  - ArabicCompiler.IDE: واجهة رسومية بسيطة لكتابة وتشغيل البرامج.
